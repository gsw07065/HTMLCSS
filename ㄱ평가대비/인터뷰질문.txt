웹표준 : ((웹에서 사용되는 기술들의 표준화))를 의미,
웹 페이지, 웹 애플리케이션을 만들 때 따라야 하는 규칙과 규격의 모음!

/사용자가 어떤 브라우저나 기기를 사용하더라도 홈페이지 화면을 동일하게 볼 수 있도록 하는 것이 웹 표준, 표준화 된 언어를 사용함으로써 동일한 결과물을 보여주는 웹 페이지 제작 기법/
           
           

웹접근성 : 모든 사용자에게 동등한 웹 콘텐츠 접근 기회를 보장하는 것

/모든 사용자가 쉽게 웹 페이지에 접근할 수 있도록 웹 접근성을 고려/
             



웹호환성 : 운영체제(OS), 브라우저에 상관없이 유사한 화면으로 동일한 정보에 접근이 가능해야 합니다. 
             다양한 환경 적용, 새로운 기기 이용
             다양한 브라우저와 디바이스에서의 웹 콘텐츠 호환성을 유지하는 데 중점


UI(user interface) : 사용자와 시스템 간의 상호작용이 원활하게 이루어지도록 도와주는 장치나 소프트웨어

UX(user experience ): UI를 통해 사용자가 제품이나 서비스에 대해 전체적으로 느끼는 사용자 경험을 의미

web safe color : 운영체제(OS) 및 브라우저의 종류에 상관없이 공통으로 사용되는 안전한 색

html : 문서 내용 지정 언어  하이퍼 텍스트 마크업 랭귀지?
CSS(Cascading Style Sheets) : Html로 작성된 문서의 서식을 지정하는 스타일시트 언어
자바스크립트 : 웹 동적구현 스크립트언어




네이티브앱 : 우리가 흔히 말하는 어플리케이션 모바일 기기에 최적화 된 언어로 개발된 앱
Android 또는 iOS 같은 어떤 구체적인 플랫폼만을 위해 만들어진 응용 프로그램

모바일웹앱(WEB+APP) : 모바일웹과 네이티브앱 결합,모바일에 최적화 된 앱

모바일웹 : 모바일에서 PC용 사이트의 글자폰트와 이미지등 데스크탑 브라우저에서 실행되는 기능을 모바일에 맞추어 표현한 사이트
             PC용 홈페이지를 모바일 스크린의 크기에 맞춰 줄여놓은것!!

하이브리드앱 : 네이티브앱 + 웹앱   - 웹개발 기술을 사용해 앱을 개발 (인스타, gmail, 금융기관)

그리드 : 페이지나 화면의 레이아웃을 구성하는 가이드라인. 요소와 간격을 균일하게 유지해 일관된 사용자 인터페이스 구현



스타일 가이드 : 일관된 브랜드 경험을 주기 위해 기본이 되어야 하는 스타일에 대한 규칙 (컬러, 서체, 모티브,아이콘스타일)
와이어프레임 : 선(Wire)'만을 사용하여 빠르게 결과물의 윤곽을 만들어내는 작업
플로우차트 : 순서도,흐름도 (처리 과정을 순서대로 나열, 도식화)
